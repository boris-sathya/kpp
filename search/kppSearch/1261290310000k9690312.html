Microsoft My first encounter :)

Recently Microsoft had come to my college to enroll people for Internship. I attended it, but was sent out in the last round. I had 3 rounds as follows.
  1. First is a written Test. (8 were shortlisted among almost 100 people)
  2. Second was a Test cum interview. (4 were shortlisted among the 8)
  3. Third is a round with 3 interviews. (You need to clear all of them to get in. 1 was selected.)
Round 1:
The first round had 4 questions.

First question. - Debugging. Given a program to find the meeting point of two linked lists I was asked to debug it. The following is the program. (I believe it is correct)

*pList FindMerge(pList list1, pList list2)
{
pList *pMover1 = list1;
pList *pMover2 = list2;

len1 = length(pMover1);
len2 = length(pMover2);

if(len1>len2)
{
while(len1 > len2) //I checked the infinite loop here
{
pMover1 = pMover1 -> Next;
}
}
else
{
while(len2 > len1) //Again an infinite loop here
{
pMover2 = pMover2 -> Next;
}
}

while(pMover1 -> Next != pMover2 -> Next)
{
/*Since it compares the next element it misses the first element on comparison. So I changed the while loop condition */
pMover1 = pMover1 -> Next;
pMover2 = pMover2 -> Next;
}

return pMover1->Next; //Changed this too
}

I got 8 on 10 for this question. I think I missed the initial NULL condition check. Lest would have got 10 I believe. :D

Second question:
You are given a circuit and you need to write a program to find the resistance between any two given points. :)
Sample input
A,B - 10
B,C - 5
etc..
Sample output: A, E- 15 ohm
You can assume your inputs and output formats.

I wrote an algorithm based on 2 D arrays. But when I was interviewed I got a few clues on how to proceed in this problem. The interviewer asked me the following.

What data structure are you reminded of when you look at this question?
I said Matrix. He expected "GRAPHS".

(I give the input as whether the two points are in series or parallel)
Do you really need that data as input? - he asked.
I told him I made that as an assumption which according to the question was allowed.
Where did you write it?
I've defined the function before I started the program.
"But assumption is different from definition" he said.
I had to accept :) But he was right.

I got 3 on 10 for that :)

Third Question:
You are given a program which takes a string input and returns a reversed string (reversed in terms of words.)

Example: Input is - I am Sornakumar.
Output is- Sornakumar am I.

I had to write test cases for this.

Its just based on experience on how you test your program. I wrote upto ten test cases like Multi line input, Multiple sentense input, special character input, number input, single word input, decimal input etc. I wrote it in the following format - Input, Expected output, Description.

I got 6 on 10. Had a satisfying interview based on this question. :)

Fourth: You are to design a universal remote control for the Indian Market. What features will you add for that?

A question to test imagination. My bad I got only 2 on 10 :(

Round 2:

This was a round where you were given a question and you will have an interview-kinda-thing based on that question.

Question:
There is a Robot which is used to clean the steps. You have two functions in the robot. gotoNextStep() and cleanTheStep(). You need to design an engineering solution for this problem.

Solution:
We were wondering what an engineering solution was. But its no big deal. Its just a practical solution. :)

I wrote the following answer:
If ROBOT is left at the top most step
It cleans the stairs from the top most step to the bottom most step.
If its left in any intermediate step
It goes all the way down doing nothing and cleans from bottom to top
Finally comes back to original position - This line makes much of a difference between an engineering solution and a programming solution.

Interview:
When you clean the stairs from the top to the bottom, the dust gets collected cummulatively to the bottom. Then why do you move down first and then clean to the top?
My Answer:
I accept what you say. Consider a 100 story building. The probability of leaving the ROBOT at the 1st story is greater than leaving it in the 100th story. As a result it would be highly inefficient to move to the top most step and start cleaning it. You will have a lot of null cycles. Moreover the effort required for the ROBOT to move down is less than the effort it has to put to move up (against gravity :D). So the solution I give is more efficient.
I also took constant values for effort and up-down movements and proved that my assumption was correct, mathematically :D

Got shortlisted. :)

Round 3: The 3-interview round as I call it :)

I had to attend 3 interviews. I was the first one to be shortlisted. So I was called first for the interview. (Disadvantage of being first :( You'll know as you complete reading. )

He told me "Do it without using queue."
Sad that I failed to remember the "First right last left" thumb rule.

Next was find inorder predecessor and further questions based on them.

Whatever program I wrote, he defeated it with a test case. Sad. The interview went very bad and it was 45 minutes of nightmare. :(

Interview 2:
Being highly demoralised by my 1st interview I stepped into my 2nd one. This time I encountered a guy who actually read my Resume. (God! That boosted some confidence.). He asked about me and my projects. I explained them all. He saw my "Area of Interest" = "Web Development". He asked me how many projects have you done in your area of interest. I said "Except 2 projects almost everything else is in my area of interest".

Question time:
He opened the SBI website Login form. Asked me to write test cases for that. The login component had two text boxes, submit, reset, and a virtul keyboard.

I wrote really nice test cases and the two which impressed him were testing against "SQL Injection, Cross site scripting attacks".

He then asked me "Why does the character alignment in the virtual keyboard keeps changing every time you refresh the site?". I said to prevent users from tracking mouse clicks. He was satisfied.

He asked me to classify all these test cases. I dint know how to do that. He then taught me about the various classifications like Security testing, Accessibility testing, functionality testing etc (Man!!!!!! He was an amazing interviewer.)

2nd Question:
Given a rectange. I cut a small rectangle in this big one to get a resultant. You need to cut the resultant to get two equal halves. Only one cut is allowed.

I gave him some answers, he told me I was almost there. But we moved on. (I leave it to you to find the answer cos I dont know it man!! :P)

5th Quesion:

I did the program. But I dint consider cases for shifting the head pointer(Insertion at the beginning.) He was satisfied though.

I came out of the interview happily. (Relief!!)

Sad moment. But I was happy that I got a lot of experience out of this. I learnt that I should think atleast twice before I utter a word in the interview. I was more satisfied than being sad. Every moment was a learning experience and I hope to do better the next time.

I came out.., friends.., traffic.., :)

HTTP Status 405 - HTTP method POST is not supported by this URL

Recently one of my friends was developing a web application. He used a simple form (with javascript validation) which fires a servlet on submit. Once I clicked on the submit button I should be forwarded to the welcome page. But this did not happen. Instead it introduced me to an unknown error "HTTP method POST is not supported by this URL".

The html code was really simple that it did not have anything to big to be debugged in it. We used eclipse. So when creating a Servlet we used "File->New->Servlet". It generated the stub and we edited it according to our need. Nothing really big to debug there also. A little googling also did not help us :(. So I decided to do it the trial and error way. I found a solution too :). Here are those results.
  • My servlet was named "kk_server". I renamed it to "kkserver". Sad that it did not solve the problem. :(
  • In the servlet which had the autogenerated stub, there was the doPost function inside which eclipse has added "super.doPost(request,response)". I commented this line and bingo!!!!!! It worked!!
I realised what naming conventions need to be followed in a web scenario. I'm not sure of what the "super.doPost()" does. But I'm really eager to find it out. Hope I find it out really soon :).

PhpMyAdmin & Stored Procedures!!

From what I've seen, PhpMyAdmin does not seem to support creating, editing or viewing Stored Procedures. So for those who are trying to do this, I should say, its better you do it the native way than to try it in PhpMyAdmin. I am using XAMPP. This article is aimed at XAMPP users who want to create Procedures in MySQL.

To open the MySQL native environment,
  • Start your XAMPP server
  • Goto Start->Run
  • Type in cmd
  • Now type cd C:\xampp\mysql\bin\ (Note: Your path may be different)
  • Type in mysql
  • You will be shown the mysql prompt.
Now all you need to do is to type in your procedure here and execute it :)
Don't worry!! I'll help you with a sample for some syntax issues :)

First you need to choose the database you are going to work with. Then type in your code. (I've a database called test)

use test;

delimiter /

CREATE PROCEDURE proctest (OUT params INT)
begin
SELECT COUNT(*) INTO params FROM test.check;
end;
/

delimiter ;

I've created a simple procedure which counts the number of entries in the table "check" in "test" database. Now I need to check if the procedure has been created or not. To do this type in,

show procedure status;

You will see the list of procedures. Now I need to check if my procedure is working or not. To do that type in,

call proctest(@samp);

select @samp;

This should display the number of rows in the "check" table :) There you go. You have your MySQL procedure working real fine :)

Triggers in PhpMyAdmin?

I've been looking for a way to create some triggers in MySQL using PhpMyAdmin. Its time I got bored on looking at the "#1064 Check your MySQL Syntax.." bla bla bla.., when I found a solution for this problem. Its real simple.

I'll take a simple code and explain my problem,

create or replace trigger trigname after insert on check
for each row
begin
insert into rows values(5);
end;

Ok.. My code has 3 problems now.

Problem 1:
From what I've experienced. I've come to know that MySQL does not support "replace" keyword. :) So my SQL is changed as follows,

create trigger trigname after insert on check
for each row
begin
insert into rows values(5);
end;

Problem 2:
I'm creating the trigger on check table and I'm not telling MySQL which database it belongs to. [:o] That is real bad about me. So I change my code as follows,

create trigger trigname after insert on test.check
for each row
begin
insert into rows values(5);
end;

Problem 3:
This is the trickiest of problems and you would face this problem only if you use some editor like PhpMyAdmin. If you notice, just below the area where you type in your SQL statement, you will find a textbox called "Delimiter". (Man,.!! I dint notice this.) This actually means that your SQL query ends with the semicolon and nothing is after that. :( So the solution is simple. Change the "Delimiter" option to some unused character. Example, enter "$$" in that textbox and change your query as follows.

create trigger trigname after insert on test.check
for each row
begin
insert into rows values(5);
end;
$$

Hurray!!! My code is error free and it works.
To see if the trigger is created or not, just give this query a try.

SHOW TRIGGERS

And this shows you the created triggers.

Configuring Eclipse, WebSphere & DB2 to work together

You have installed Eclipse, WebSphere & DB2. You have also configured Eclipse and WebSphere to work together. Now its time for us to make all these 3 softwares to work hand in hand. After all, when you create a website you will surely want your web server to access the back end. Here are a few steps on how you do it.

Creating a Database Pool:
The first step would be creating a Database pool in WebSphere Application Server. The Database Pool helps websphere to identify your database. To do this,
  1. Turn on your server. ( This can be done by going to eclipse also )
  2. Now goto your browser and type in "http://localhost:8080/"
  3. On the left pane select "Database Pools"
  4. In the page whichappears, choose "New" to create a new database pool.
  5. Now select the database you are using. In my case, I choose DB2.
  6. Now click "Next"
  7. You see a page, which has a lot of options.
  8. The first field asks you to choose the driver. Choose the driver corresponding to your DB2 version. In my case I choose v9.5, db2jcc.jar and v9.5, db2jcc_lisence_ui.jar
  9. Press ctrl to select multiple options in the field.
  10. Now fill in the required details like username, password, Database name, port number etc.
  11. Click on next.
  12. Your Database pool is created.
Configuring your project with this pool:
The next step would be to configure your project to work with this Database pool.
  1. Now that you have created your database pool, goto "http://localhost:8080/" and choose "Database pools" in the left pane.
  2. You will see your new Database pool listed.
  3. Corresponding to that entry, you will find a link called "Usage".
  4. Click on this link.
  5. This link will show you the xaml code which helps in connecting your project and the Database pool.
  6. Do as it is instructed in that page.
  7. You will have to modify you "web.xml" file and "geronimo-web.xml" files.
  8. Copy and paste the corresponding code sections.
  9. You should see no errors once you have edited the xml files.
  10. If you dont see any errors then your Project is configured :)

In case you see a run time error, then this might be your problem. Your XML attributes are wrong. A simple examle would be this,

Your tag will look like this,

<dependencies ...

instead of,

<dep:dependencies ...


So you will have to change these.. The following are how the tags will look like.

<dep:moduleId ...
<dep:dependencies ...
<name: resource-ref ...


Their closing tags should also have the same prefixes. :)
Hope it works after changing these :)

Eclipse - Ur program is not running? :o

You would have typed in a complete Java program in your eclipse, and when you run, you might find your previous program which had main function in it running all over again. It happened for me too. Firefox - Cache, a Disadvantage?

Recently I've been working with a lot of Web-related projects. When I test-run my projects I go for my obvious choice, the Firefox. All my Web Servers in the localhost have been configured to open my projects by default in Firefox. Recently I discovered a serious disadvantage with Firefox. This is how it happened.

I was working with my project. My internet got disconnected suddenly. I dint notice that. As usual, to test my project, I opened my Firefox and deployed my project using my server. I found some output which was not the one which was expected. So I modified some code. Tried running in Firefox again. Had the same result. Nothing changed. I dint know what the problem was. I tried coding it all over. Nothing changed. I restarted my server. Firefox refused to show a difference. I even deleted my server and configured it again. Tried running the project. The result was the same. I had no clues on what had happened. I finally decided to quit. So clicked on the File menu and was about to choose "Quit", when something stuck me. The Work Offline checkbox in the File menu was checked. Man I understood the reason for the malfunction.

My system has got disconnected from the Internet and my Firefox has migrated to "Work Offline" mode. So whatever I visited came from its cache and not the result of my Web server's deployment. Sad. I unchecked it and it worked fine. :) So the next time you have one such problem make sure that you check this Work-Offline option first. The cache has been a serious disadvantage man!!

What's best? Cookies or Sessions?

I've been wondering for a while on which method can help me best identify a user once he has logged in to my site. I found the use of Sessions and Cookies really interesting and useful. But which one do I go with. Both of them achieve the same goal, but which one is better. Well, if you ask me, it depends on the situation. Here are a few things, I felt, could distinguish them!

COOKIES:
  • If you are creating a site where you want to identify the user the next time he logs in, you can well use a cookie.
  • If you are not really bothered about how the user browses your website (the state), then you are a welcome customer for Cookies. Of course., states can be maintained using cookies too, but it involves obvious additional work.
  • The MOST IMPORTANT point: Cookies are stored on the client. The client may not support cookies. In such situations SESSIONS are your only go.
  • Moreover when you store data on the client, you give more control to the client. So if you are building an ultra-secure website, cookies are generally not recommended.

SESSIONS:

First let me tell you how SESSIONS work.They work in the same way as the cookies, but the only difference is they get stored in the server and not in the client. Only the session ID gets stored in the client. This session ID is stored in a cookie. If the client does not support cookies, then the session ID is embedded within the URL. Here are some of its advantages.
  • You provide limited control to the user.
  • If you are interested in a site, where you are really bothered on the state of the user, then sessions can be your best deal.
  • No problems with time will occur (as I mentioned in some previous posts) as in Cookies. So you can reduce on thinking about these disadvantages.
A mixed use of these can also be done. This again depends on our need!

PHP - Delete cookie?

I searched around a bit, but dint find any explicit function to delete a cookie. Most of the methods seem to do the same thing, EXPIRE the cookie. They set the time to the past and thus the cookie gets expired and is usable no more. Here is how it works.

setcookie( "cookieid", "", time() - 3600);

I set it back by one hour with null value to the cookie! Its gone! But wait. There is yet another catch in this. The 'time()' function returns the time in the server. Say 3:30(Current time). It sets it back by 1 hour. So the effective value is 2:30. Cookie expires at 2:30(exp time). That is, Server asks the client to make the cookie expire at 2:30.

One important property of cookie is that its local to your PC! So all the values of time associated with this cookie is your local PC's time. In case that your local PC's clock does not work or it has stopped working at 1:30, then the cookie does not expire. That is, Server asks you to expire your cookie at 2:30, but your time stands still at 1:30 and thus the cookie never gets expired. This is a serious problem. There is a solution. Instead of giving a negative time, you can give in 1. using a time of ‘1′ indicates an expiration time of 1 second after midnight, January 1st, 1970 which is the earliest possible expiration time. (But beware. If the client's clock showed some value before that date, then no one can be responsible :) ). The function should look like this.

setcookie( "cookieid", "", 1);

I was trying out cookies in my PC. I was puzzled with some results. I was designing a small website which has a login and member's area. When the user clicks on "Logout", it redirects him to "logout.php". In "logout.php", I used the above function to clear my cookies. And in the next line I tried to print the value of a cookie. This is how it looks:

..
setcookie("locuser", "", time() - 3600);
setcookie("locpass", "", time() - 3600);
echo $_COOKIE["locuser"];
..

(I am using "localhost". So temporarily I am not bothered on the time problem. :) )
I should ve got no outputs. But instead it showed me the value of the cookie. :(

I redirected the resulting page to some other page and printed the cookie. It dint show up anyhing. So what should have been the possible problem? I think there can be 2 perspectives
  • The cookies get set only when you leave the page where you type in the "setcookie" function.
  • Or when the page gets loaded, only the cookie-related fuctions are executed only at the end!
Well, I'm not sure on what was going on. But should find out soon.

PHP count the number of occurrences of a character in a string

This is really a dumb information. But still it can be useful at times. I was coding a website in PHP today. I had a situation when I needed to count the number of occurrences of a character in a string. I was searching for some inbuilt function to help me out. At the first instance I dint find one.(Atleast that is what I thought). But the truth was, I missed one. There is a function which is used to count the number of occurrences of a substring in a string. Damn!! It dint strike me!! I used it and BOOM! I got the needed result.

Here is the simple code.

substr_count($variable, "l");

Here, I am searching for the number of occurrences of "l" in the string "variable".

If it was not for my ignorance, I would have saved some 20 minutes of my searching time :(.
