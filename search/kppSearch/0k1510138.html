There are several sources for Linux Information. Mailing Lists, Newsgroups, the Linux Documentation Project, books, etc.

This page lists some of the more common and interesting sources for information on Linux.

  • Linux Online
    The first stop for Linux information on the Internet. Here you can find anything from usergroups in your town to useful books and documentation.

  • HOWTO's
    Linux HOWTO and mini-HOWTO documents. There are detailed documents on specific subjects. A must-read for setting up various parts of Linux and Linux applications.

  • Linux Online Guides
    Linux Guides, including Linux Administrators Guide, User's Guide, Network Adm Guide, The Linux Kernel and a System Admin Guide. [Local mirror | LDP site]

  • Linux Gazette
    Linux Gazette, a member of the Linux Documentation Project, is an on-line WWW publication that is dedicated to two simple ideas: Making Linux just a little more fun and Sharing ideas and discoveries.
