gleem is a small, self-contained library of 3D widgets that support direct user interaction with a 3D scene. Java and C++ programming language versions are available. The "official" gleem home page is http://www.media.mit.edu/~kbrussel/gleem/. The included documentation is on-line.

Download gleem

There are two source code packages available:

  • http://ftp.gnu.org/pub/gnu/gleem/gleem-1.2.1.tar.gz (225k) - source code only (Unix format)
  • http://ftp.gnu.org/pub/gnu/gleem/gleem-1.2.1.zip (352k) - source code only (Windows format)

Precompiled Java class files are also available:

  • http://ftp.gnu.org/pub/gnu/gleem/gleem-1.2.1.jar.gz (87k) - compiled .class files only (jar + gzip format)
  • http://ftp.gnu.org/pub/gnu/gleem/gleem-1.2.1.jar.zip (87k) - compiled .class files only (jar + zip format)
